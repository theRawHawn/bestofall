# Custom Logos Directory

Place your manual logo images (`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`) in this folder.

The app is set to look for the following file names:
- QuickBooks: `qickbooks_logo.jpeg` (or `quickbooks_logo.jpeg`)
- SAP ERP: `sap_logo.jpeg` (or `sap_logo.png`)
- Tally Prime: `tally_prime_logo.jpeg` (or `tally_prime_logo.png`)
- Zoho Books: `zoho_books_logo.jpeg` (or `zoho_books_logo.png`)

If the images are placed here, they will automatically override the SVG vectors and render gracefully!
